# Request a Feature / *Richiedi una Funzionalita'*

Use this form to suggest improvements to DRUCS — the web dashboard, mobile app, firmware, or documentation. Your request will be reviewed by the development team.

*Usa questo modulo per suggerire miglioramenti a DRUCS — dashboard web, app mobile, firmware o documentazione. La tua richiesta sara' esaminata dal team di sviluppo.*

---

!!! info "No GitHub account needed / *Non serve un account GitHub*"
    This form creates a GitHub Issue automatically. You don't need a GitHub account.

    *Questo modulo crea automaticamente una Issue su GitHub. Non serve un account GitHub.*

---

## Feature Request Form / *Modulo Richiesta Funzionalita'*

<div class="feature-form" markdown>

**Your Name / *Il Tuo Nome*** *

> *Example: Berry Boertien*

**Product / *Prodotto*** *

> Select: BravoCompact / Bravo / Smart / SilentTrack / VipAir / Web Dashboard / Mobile App / General

**Priority / *Priorita'*** *

> Select: Nice to have / Important / Critical for my work

**What do you need? / *Di cosa hai bisogno?*** *

> Describe the feature or improvement you'd like. Be as specific as possible.
>
> *Descrivi la funzionalita' o il miglioramento che vorresti. Sii il piu' specifico possibile.*

**Why is this useful? / *Perche' e' utile?*** *

> How would this help you or your customers?
>
> *Come aiuterebbe te o i tuoi clienti?*

**Screenshots or examples / *Screenshot o esempi*** (optional / *opzionale*)

> Attach any images, sketches, or links that help explain your request.
>
> *Allega immagini, schizzi o link che aiutino a spiegare la tua richiesta.*

</div>

## What Happens Next / *Cosa Succede Dopo*

1. Your request is logged as a GitHub Issue / *La tua richiesta viene registrata come Issue GitHub*
2. The team reviews and prioritizes it / *Il team la esamina e la assegna una priorita'*
3. You'll be notified when work starts and when it's released / *Sarai notificato quando inizia il lavoro e quando viene rilasciata*
4. Check the [Roadmap](../roadmap/index.md) to see planned features / *Controlla la [Roadmap](../roadmap/index.md) per vedere le funzionalita' pianificate*
