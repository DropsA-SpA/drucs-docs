# Report an Issue / *Segnala un Problema*

Use this page to report bugs, malfunctions, or unexpected behavior in DRUCS.

*Usa questa pagina per segnalare bug, malfunzionamenti o comportamenti inattesi in DRUCS.*

---

## Before Reporting / *Prima di Segnalare*

Check these common solutions first / *Controlla prima queste soluzioni comuni:*

!!! tip "Device not connecting to cloud / *Dispositivo non si connette al cloud*"
    1. Verify WiFi is enabled (setting #40000) / *Verificare che il WiFi sia abilitato*
    2. Check SSID and password / *Controllare SSID e password*
    3. Ensure 2.4 GHz network (5 GHz not supported) / *Assicurarsi rete 2.4 GHz*
    4. Check device voltage (>20V required) / *Controllare tensione dispositivo (>20V necessari)*

!!! tip "Dashboard shows 'Offline' / *La dashboard mostra 'Offline'*"
    - Device may have lost WiFi connection / *Il dispositivo potrebbe aver perso la connessione WiFi*
    - Check the 'Last Seen' time on the dashboard / *Controllare l'orario 'Ultimo Contatto' nella dashboard*
    - If >24h: the device is likely powered off or disconnected / *Se >24h: il dispositivo e' probabilmente spento o disconnesso*

!!! tip "Alarm won't clear / *L'allarme non si cancella*"
    - Fix the root cause first (e.g., refill reservoir) / *Risolvere prima la causa (es. riempire il serbatoio)*
    - Then send "Reset Alarms" command from dashboard or app / *Poi inviare il comando "Resetta Allarmi" dalla dashboard o app*
