# Smart 4.0

<span class="badge badge-sm">SM</span> **Compact lubricator with IO-Link / Lubrificatore compatto con IO-Link**

---

## Overview / Panoramica

The Smart 4.0 is a compact lubrication unit with IO-Link connectivity for Industry 4.0 integration. It shares the core feature set with BravoCompact but adds IO-Link for machine integration.

*Lo Smart 4.0 e' un'unita' di lubrificazione compatta con connettivita' IO-Link per l'integrazione Industria 4.0. Condivide le funzionalita' base con BravoCompact ma aggiunge IO-Link per l'integrazione macchina.*
