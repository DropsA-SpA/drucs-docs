    How to Request Features / Come Richiedere Funzionalita'¶

    Field engineers and stakeholders can request features through the Feature Request page. No GitHub account needed.

    I tecnici sul campo e gli stakeholder possono richiedere funzionalita' tramite la pagina Richiesta Funzionalita'. Non serve un account GitHub.
